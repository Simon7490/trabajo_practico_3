Java

¿Qué es Java?

Java es un lenguaje de programación de alto nivel y una plataforma informática desarrollada por Sun Microsystems. El lenguaje fue lanzado por primera vez en 1995 y ha evolucionado con el tiempo para convertirse en uno de los lenguajes de programación más populares en el mundo.

Características de Java

  • Orientado a objetos: Todo en Java se organiza en torno a objetos y clases, lo que facilita la reutilización de código y la modularización.
  • Independiente de la plataforma: El código Java se compila a bytecode, que puede ejecutarse en cualquier máquina virtual Java (JVM), independientemente del hardware subyacente.
  • Multitarea: Java tiene soporte incorporado para la creación y gestión de múltiples hilos de ejecución.
  • Seguro y robusto: Java proporciona mecanismos de manejo de excepciones y verificación de código en tiempo de ejecución para aumentar la seguridad y robustez del programa.

Ventajas de Java

  • Portabilidad: El lema "Write Once, Run Anywhere" (Escribir una vez, ejecutar en cualquier lugar) se refiere a la capacidad de Java de ejecutarse en cualquier plataforma que tenga una JVM.
  • Rendimiento: Aunque Java es un lenguaje interpretado, el uso de compiladores Just-In-Time (JIT) mejora significativamente su rendimiento.
  • Bibliotecas Ricas: Java tiene una extensa biblioteca estándar que proporciona muchas funcionalidades listas para usar, desde estructuras de datos hasta bibliotecas de redes y gráficos.
  • Comunidad y Soporte: Java tiene una gran comunidad de desarrolladores, lo que facilita encontrar recursos, tutoriales y soporte.

HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. HTML describe la estructura de una página web y sus elementos.

Estructura Básica de HTML

Un documento HTML comienza con una declaración de tipo de documento (DOCTYPE) que especifica la versión de HTML. La estructura básica de un documento HTML incluye etiquetas como <html>, <head>, y <body>.

Estructura Básica de HTML


						!DOCTYPE html
						html lang="es"
						head
						    meta charset="UTF-8"
						    titleTítulo de la Página/title
						/head
						body
						    h1Encabezado Principal/h1
						    pEsto es un párrafo./p
						/body
						/html

Etiquetas Comunes de HTML

  • <h1> a <h6>: Encabezados, donde <h1> es el más importante y <h6> es el menos importante.
  • <p>: Párrafo de texto.
  • <a>: Enlace de hipervínculo.
  • <ul>, <ol>, <li>: Listas desordenadas, ordenadas y elementos de lista.
  • <div>: División o sección de contenido.
  • <span>: Contenedor en línea para texto.
  • <img>: Imagen.

Atributos Comunes de HTML

Las etiquetas HTML pueden tener atributos que proporcionan información adicional sobre el elemento. Algunos atributos comunes son:

  • id: Identificador único del elemento.
  • class: Clase del elemento, utilizado para aplicar estilos CSS.
  • src: Fuente del archivo (utilizado con <img>).
  • href: URL del enlace (utilizado con <a>).
  • alt: Texto alternativo para imágenes.
