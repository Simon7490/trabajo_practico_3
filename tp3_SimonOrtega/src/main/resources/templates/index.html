Teoría

Programación

“El término programación se define como un conjunto de instrucciones consecutivas y ordenadas que llevan a ejecutar una tarea específica. Dichas instrucciones se denominan 'código fuente', el cual es único para cada lenguaje y está diseñado para cumplir una función o propósito específico.” (Morales, 2014)

Lenguaje de Programación

Un lenguaje de programación es una “estructura que, con una cierta base sintáctica y semántica, imparte distintas instrucciones a un programa de computadora” (Perez y Merino, 2009).

Un lenguaje de programación está compuesto por:

  • **Símbolos**
  • **Reglas sintácticas y semánticas**
  • **Estructura**
  • **Significado**
  • **Elementos y expresiones**

IDE (Entorno de Desarrollo Integrado)

Los IDE (Integrated Development Environment) son entornos de desarrollo donde es posible escribir, compilar (o interpretar) y ejecutar código en un lenguaje de programación, todo en el mismo programa.

Estos entornos facilitan el desarrollo de aplicaciones al incorporar:

  • **Herramientas de Debug:** tanto textual (mediante una consola) como gráfica.
  • **Bibliotecas:** componentes ya desarrollados para ser utilizados en el programa.
  • **Interfaz gráfica:** que facilita y agiliza la escritura de los programas.
  • **Funcionalidades adicionales:** acceso y gestión de bases de datos, integración con herramientas de control de versiones, simulación de servidores, generación de documentación, gestión de interfaces de usuarios, modelado de artefactos, etc.

Java

**¿Qué es Java?** Java es una tecnología que incluye un lenguaje de programación y una plataforma de desarrollo.

**La plataforma Java** Por definición, una plataforma es el ambiente hardware o software sobre el cual se puede ejecutar un programa.

  • **Plataforma Windows:** Un PC con su sistema operativo Windows.
  • **Plataforma Linux:** Una notebook con una distribución Linux.
  • **Plataforma Android:** Un celular con Android.

La plataforma Java se diferencia porque es una plataforma software que se ejecuta sobre las prestaciones que brinda otra plataforma que gestiona el hardware (plataforma base).

La plataforma Java tiene dos componentes principales:

  • **Java Virtual Machine (JVM):** Es un intérprete que traduce el bytecode al código máquina de la plataforma base al momento de ejecutar la aplicación.
  • **Java Application Programming Interfaces (API):** Una colección de componentes software organizados en bibliotecas (packages) que proveen muchas capacidades útiles para el desarrollador.

**El lenguaje de programación Java** Java es un lenguaje de programación de alto nivel definido por sus características: sencillo, orientado a objetos, distribuido, interpretado, robusto, seguro, independiente de las arquitecturas, portable, eficaz, multitarea y dinámico.
